# roger.ai client

This is a Vite-based React + TypeScript app for the roger.ai TODO agent.

## Layout

├─ client/
│  ├─ README.md
│  ├─ Dockerfile
│  ├─ package.json
│  ├─ tsconfig.json
│  ├─ index.html
│  ├─ vite.config.ts
│  └─ src/
│     ├─ main.tsx
│     ├─ App.tsx
│     ├─ api.ts
│     ├─ types.ts
│     ├─ styles.css
│     └─ components/
│        ├─ ProjectSelector.tsx
│        ├─ TodoForm.tsx
│        └─ TodoList.tsx

## Running locally

To run locally,

```bash
npm install
npm run dev
```
